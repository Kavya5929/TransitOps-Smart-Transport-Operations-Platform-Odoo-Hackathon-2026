# 🚛 TransitOps – Smart Transport Operations Platform

> A modern transport and fleet management platform built for the **Odoo Hackathon** to simplify vehicle operations, maintenance, fuel management, safety monitoring, and reporting through a centralized dashboard.

## 🌐 Live Demo

🔗 https://transitops-6fc9.onrender.com

---

# 📌 Overview

TransitOps is an all-in-one fleet management solution designed to help transport companies efficiently manage their daily operations.

The platform provides dedicated modules for:

- 🚚 Vehicle Management
- 👨‍✈️ Driver Management
- 🛣️ Trip Management
- ⛽ Fuel Monitoring
- 🔧 Maintenance Planning
- ⚠️ Safety Monitoring
- 💰 Expense Tracking
- 📊 Reports & Analytics

The goal is to reduce manual work, improve operational efficiency, and provide real-time insights into transport activities.

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/transitops.git
```

---

## Backend Setup

```bash
cd backend
npm install
```

Create a `.env` file:

```env
PORT=5000
MONGODB_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
```

Run backend:

```bash
npm run dev
```

---

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

# 🎯 Problem Statement

Managing transport fleets manually often leads to:

- Poor maintenance scheduling
- Fuel misuse
- Untracked expenses
- Delayed reporting
- Safety compliance issues

TransitOps centralizes these operations into one intelligent platform, enabling transport teams to monitor, manage, and optimize fleet performance efficiently.
